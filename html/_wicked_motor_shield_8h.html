◆ M1_BRAKE_MASK

#define M1_BRAKE_MASK   (0x10)

If M1_BRAKE_MASK bit is zero, brake is set to BRAKE_OFF and M1_DIR_MASK bit indicates direction of rotation (Bit is set for DIR_CW and cleared for DIR_CCW). If M1_BRAKE_MASK bit is one, M1_DIR_MASK indicates whether hard brake (direction bit is set) or soft brake (direction bit is cleared).

This means that clearing the brake bit (location indicated by M1_BRAKE_MASK) means that the direction bit (indicated by M1_DIR_MASK ) must be reloaded using the value of #old_dir .

When setting the brake bit (location indicated by M1_BRAKE_BIT ), the value of the direction bit must be copied to #old_dir .
